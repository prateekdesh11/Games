#  Games Collection

A collection of four classic Python games created by Prateek.  
Each game runs in the terminal and requires only Python 3 — no extra libraries.

---

##  Included Games

- **BlackJack.py** — Interactive Blackjack with ASCII cards, twist/hold logic, dealer AI, and a two‑player mode.
- **Bridge.py** — Simplified Bridge trick‑taking game where you play as South against three computer players.
- **Hangman.py** — Hangman with a big list of tricky English words and full ASCII art stages.
- **Connect 4.py** — Two‑player Connect 4 with horizontal, vertical, and diagonal win detection.

---

## How to Run

In a terminal, inside the project folder:

```bash
python "BlackJack.py"
python "Bridge.py"
python "Hangman.py"
python "Connect 4.py"
